# Task: 프론트엔드 기반 설정

## Description

Tailwind CSS, Vite, Vue.js 등 프론트엔드 기반 환경을 설정합니다. 디자인 시스템의 컬러 팔레트, 타이포그래피, 그림자 등을 Tailwind 설정에 반영합니다.

## Acceptance Criteria

- [ ] Tailwind CSS 3.x 설치 및 설정 완료
- [ ] Vite 5.x + Vue.js 3.x 플러그인 설정 완료
- [ ] Pinia, Axios, Lodash 패키지 설치 완료
- [ ] 디자인 시스템 컬러 팔레트 (Primary #FF385C) Tailwind 설정에 반영
- [ ] 커스텀 폰트 (Pretendard) 설정 완료
- [ ] 커스텀 그림자, border-radius Tailwind 확장 완료
- [ ] Axios 인터셉터 (CSRF 토큰, 401 핸들링) 설정 완료

- [ ] `npm run dev` / `npm run build` 정상 동작
