## Technical Details

### 파일 구조
```
resources/views/components/layouts/
├── app.blade.php          # 메인 레이아웃
├── header.blade.php       # 헤더
├── footer.blade.php       # 푸터
└── mobile-nav.blade.php   # 모바일 하단 네비
```

### 헤더 레이아웃 (데스크톱)
```
┌─────────────────────────────────────────────────────────┐
│  [Logo]    [검색바 - 위치/날짜/인원]    [언어▼] [로그인]  │
└─────────────────────────────────────────────────────────┘
```

### 반응형 브레이크포인트
- Mobile: 0 ~ 767px (하단 네비게이션)
- Tablet: 768px ~ 1023px (간소화된 헤더)
- Desktop: 1024px+ (풀 헤더)

### 스크롤 동작
```javascript
// 스크롤 시 헤더 배경 변경
window.addEventListener('scroll', () => {
  const header = document.querySelector('header');
  if (window.scrollY > 50) {
    header.classList.add('bg-white', 'shadow-md');
  } else {
    header.classList.remove('bg-white', 'shadow-md');
  }
});
```
