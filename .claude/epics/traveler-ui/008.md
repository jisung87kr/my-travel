## Technical Details

### 파일
```
resources/views/traveler/my/
├── index.blade.php      # 마이페이지 메인 (예약 내역)
├── bookings.blade.php   # 예약 내역
├── wishlist.blade.php   # 위시리스트
├── reviews.blade.php    # 리뷰
└── profile.blade.php    # 프로필
```

### 예약 카드 레이아웃
```
┌─────────────────────────────────────────────┐
│  ┌────┐  전주 한옥마을 당일투어              │
│  │Img │  2024.12.25 (수) · 성인 2, 아동 1   │
│  └────┘  예약번호: BK-2024-001234           │
│          상태: [확정]                        │
│          [상세보기] [취소하기] [리뷰쓰기]    │
└─────────────────────────────────────────────┘
```

### API 연동
- `GET /api/my/bookings` - 내 예약 목록
- `GET /api/my/bookings/{id}` - 예약 상세
- `DELETE /api/my/bookings/{id}` - 예약 취소
- `GET /api/my/wishlist` - 위시리스트
- `DELETE /api/my/wishlist/{id}` - 위시리스트 삭제
- `GET /api/my/reviews` - 내 리뷰 목록
- `POST /api/reviews` - 리뷰 작성
- `PUT /api/my/profile` - 프로필 수정

### Vue 컴포넌트
- `WishlistButton.vue` - 위시리스트 토글 (하트 애니메이션)
- `ReviewForm.vue` - 리뷰 작성 폼
