### 레이아웃 구조
```
┌─────────────────────────────────────────────┐
│  Header (Sticky)                             │
├─────────────────────────────────────────────┤
│  필터 바 (Sticky)                            │
│  [카테고리▼] [가격▼] [날짜▼] [지역▼] [더보기]│
├──────────────────────┬──────────────────────┤
│  상품 그리드         │  지도 (토글)          │
│  - 반응형 그리드     │  - 마커 클러스터      │
│  - 무한 스크롤       │  - 가격 표시          │
└──────────────────────┴──────────────────────┘
```

### 무한 스크롤 구현
```javascript
const observer = new IntersectionObserver((entries) => {
  if (entries[0].isIntersecting && hasMore) {
    loadMoreProducts();
  }
});
observer.observe(document.querySelector('#load-more-trigger'));
```

### API 연동
- `GET /api/products?page=1&category=tour&region=seoul&...`
- Lodash debounce로 필터 변경 시 API 호출 최적화
