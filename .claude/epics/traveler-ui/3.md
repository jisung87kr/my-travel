# Task: 공통 UI 컴포넌트

## Description

에어비앤비 스타일의 재사용 가능한 Blade UI 컴포넌트를 생성합니다. Modal, Toast, Dropdown, Alert, Button, Input, Card, Badge, Skeleton, Pagination 등 전체 앱에서 사용할 기본 UI 요소를 구현합니다.

## Acceptance Criteria

- [ ] Modal 컴포넌트 (열기/닫기 애니메이션, 백드롭)
- [ ] Toast 컴포넌트 (성공/에러/경고 타입, 자동 사라짐)
- [ ] Dropdown 컴포넌트 (위치 자동 조정, 키보드 네비게이션)
- [ ] Alert 컴포넌트 (인라인 알림, 닫기 버튼)
- [ ] Button 컴포넌트 (primary/secondary/ghost 변형, 크기, 로딩 상태)
- [ ] Input 컴포넌트 (라벨, 에러 메시지, 아이콘)
- [ ] Select 컴포넌트 (기본 스타일링)
- [ ] Card 컴포넌트 (호버 효과, 그림자)
- [ ] Badge 컴포넌트 (상태별 색상)
- [ ] Skeleton 컴포넌트 (로딩 플레이스홀더)
- [ ] Pagination 컴포넌트 (Laravel 페이지네이션 호환)
- [ ] Avatar 컴포넌트 (이미지/이니셜 폴백)

## Technical Details

### 파일 구조
```
resources/views/components/ui/
├── modal.blade.php
├── toast.blade.php
├── dropdown.blade.php
├── alert.blade.php
├── button.blade.php
├── input.blade.php
├── select.blade.php
├── card.blade.php
├── badge.blade.php
├── skeleton.blade.php
├── pagination.blade.php
└── avatar.blade.php
```

### 컴포넌트 사용 예시
```blade
<x-ui.button variant="primary" size="lg">예약하기</x-ui.button>
<x-ui.modal name="login-modal">...</x-ui.modal>
<x-ui.toast type="success" message="예약이 완료되었습니다" />
```

### 스타일 가이드
- Primary 버튼: `bg-primary-500 hover:bg-primary-600`
- 카드 그림자: `shadow-card hover:shadow-card-hover`
- 둥근 모서리: `rounded-xl` (12px)
- 트랜지션: `transition duration-200 ease-in-out`
