# PRD: 제공자(Vendor) UI

## Executive Summary

관광 상품 제공자를 위한 대시보드 UI를 구현합니다. 상품 등록/관리, 예약 관리, 일정/재고 관리, 고객 커뮤니케이션 등 사업 운영에 필요한 모든 기능을 직관적인 인터페이스로 제공합니다.

**디자인 원칙:**
- 데이터 중심의 대시보드 레이아웃
- 효율적인 워크플로우 (최소 클릭)
- 실시간 업데이트 (예약 알림)
- traveler-ui와 일관된 디자인 시스템 공유
